Frostbite v0.1

Features
	- Clones on restart/login/lock, restores on unlock
	- 1-click .bat installers with auto-Admin elevation
	- Internal .ps1 scripts auto-hidden post-install

Fixes
	- Fixed Task Scheduler XML strict schema validation errors
	- Optimized triggers (Logon/Lock/Unlock) for reliable display driver readiness
	- Added detached background job for 100% clean uninstallation
	- Github release support

Usage
	- Install: Extract zip -> Run install_frostbite.bat
	- Uninstall: Run UNWISE.bat from C:\Frostbite
